# nlp_all_labs_to_submit
All mandatory NLP labs

* Lab 2 --> Hello World Transformers
* Lab 3 --> Transformers for Text Classification
* Lab 4 --> Clustering and Topic Modeling
* Lab 5 --> Classification with Generative Models
* Lab 6 --> Finetuning Ollama

To run these labs, please use GoogleCollab and select GPU-T4 to run the notebook.
Please also note that some labs necessitate you to have a Groq API key or a Hugging Face token.
In this case, change the key name to your secret key name.
